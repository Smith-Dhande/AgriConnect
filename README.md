# AgriConnect

A platform that connects farmers, buyers, and agricultural services because apparently WhatsApp groups and word-of-mouth weren't chaotic enough.

> Built to make agriculture smarter, faster, and slightly less dependent on "my neighbor said so."

## Features

* Direct buyer-farmer connections
* Agricultural product marketplace
* Real-time crop and market information
* Modern responsive UI
* Search and filtering system
* Secure authentication
* Dashboard for managing listings
* Mobile-friendly experience

## Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeather Api
* Government API for market Rates 

## Why This Exists

Farmers often struggle to find the right buyers.

Buyers struggle to find reliable suppliers.

Middlemen somehow find a way to appear anyway.

AgriConnect attempts to make the entire process more transparent by bringing everyone onto a single platform.

Will it solve every agricultural problem?

Probably not.

Will it reduce the number of phone calls that start with "Bhai rate kya chal raha hai?"

Hopefully.
